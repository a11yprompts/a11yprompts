# a11yprompts
Accessibility rules and prompts for AI coding assistants. WCAG-grounded, tool-agnostic, built for Claude Code, Cursor, Copilot, and more.

## Rule File Frontmatter Schema

```yaml
---
id: component-level          # e.g. images-a, forms-aa
title: Component — Level     # e.g. Images — Level A
component: component-name    # e.g. images, forms, buttons
level: A | AA | AAA | best-practices
tags: [tag1, tag2]           # component name + relevant keywords
---
```
